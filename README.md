# mango-test
## Getting started
```
cd docker
```
## Init docker
```
docker compose build
```

```
docker compose up
```
## Install npm packages
```
npm install
```
## Run migrations

```
npm run migration:run
```

```
npm run start:debug
```

## Open swagger
http://127.0.0.1:3000/api/admin

## Open swagger stats
http://127.0.0.1:3000/swagger-stats/ui#sws_summary
